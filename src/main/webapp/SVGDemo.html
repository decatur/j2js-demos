SVG Demo

The demo uses the org.w3c.dom.svg API to plot circles onto a canvas.

This demo requires either

  • Firefox 1.5 or higher with native SVG support,
  • Internet Explorer 6.0 or higher with Adobe SVG Viewer 3.0 or higher,
  • Opera 9.0 or higher with native SVG support, or Safari 3 or higher.

It is implemented in SVGDemo.java.
